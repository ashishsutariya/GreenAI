Green AI

GreenAI is an AI computer vision model created by high school students to make recycling easy and accessible to all.

By creating a computer vision/AI model that can detect waste materials and guide users in proper disposal, we can empower individuals and communities to make informed choices, improve recycling rates, reduce waste pollution, and contribute to a more sustainable future.

As students we created this project to expand our knowledge of computer vision and AI models, and integrate AI with day-to-day activities, like recycling.

What is Green AI? Try Our Model

Built with the power of Roboflow, Bootstrap, and many other tools.

GreenAI is a project built for the purpose of learning AI and experimenting with computer vision. The project is a simple web app hosted on Replit that uses the Roboflow classification model through the Roboflow API to classify images into 6 different recycle categories: Plastic, Metal, Paper, Glass, Compost, and Recycling Codes.

The model was trained on roughly 2,200 raw images of recyclable material, and strengthened with data augmentation, making our dataset roughly 4,500 images.

We trained this model using Roboflow, a program where users can organize and label their data by annotating or using bounding boxes. The structure and backend of the web app was built using HTML, Javascript, and Flask. The visual elements and front end of the web app were built using Bootstrap 5, and CSS.

We began by utilizing a modified script from SerpAPI to scrape images from Google Images and to create our first dataset. Because this didn't give us the best data, we went to other platforms such as HuggingFace.co and Roboflow Universe to collect better data for our model.
We then used Roboflow to classify the images into six categories: Plastic, Metal, Paper, Glass, Compost, and Recycling Codes. After some data augmentation, we trained our model on Roboflow, and received a model with accuracy of 95.4%. The trained model was hosted on Roboflow, and the web app was built using Flask, using the model and Roboflow's inference APIs to make calls to the model.

The Context

Roughly 32 percent of Americans recycle daily since 1960, while only 9 percent of the world's materials are recycled. Despite the percentages increasing monthly, many citizens lack education about recycling, do not have access to recycling, and have a lack of concern. We took it upon ourselves to create a tool that makes recycling simple, a tool that can educate and empower communities to care for the environment. Now, instead of having to do research into the material of your waste, you can simply scan the object you wish to dispose of and get simple instructions on how to properly dispose of it.
